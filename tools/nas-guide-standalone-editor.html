폴더를 선택하면 하위 폴더/파일을 추가할 수 있고, 파일은 이름만 수정 가능합니다.

사용 방법

1. 왼쪽 트리에서 항목 선택
2. 오른쪽에서 이름 수정 / 폴더 추가 / 파일 추가
3. 현재 상태로 HTML 저장 버튼으로 저장본 생성
4. 그 HTML 파일을 다시 열면 이어서 수정 가능

나중에 이 파일을 Codex/Hermes에 주면 구조를 읽어서 nasGuide 페이지에 옮기기 쉽게 되어 있습니다.
