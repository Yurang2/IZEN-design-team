# Naming Rules

## 기본 요소
- 브랜드: `IZEN`, `IAM`, `ZENEX`, `Cleanimplant`
- 콘텐츠명: 한영 혼용 가능, 공백 대신 하이픈(`-`)
- 언어: `EN`, `RU`, `ZH`, `KO`
- 규격: 인쇄 `A3`, `A4`, 디지털 `1080x1080`, `16x9`

## 버전 체계
- PROJECT 소스파일: `v01`, `v02`, `v03`
- Google Drive 배포본: `Rev01`, `Rev02`
- `_작업중` 표시는 PROJECT 안에서만 허용

## 대표 형식
```text
IZEN_CIS2026_리플렛_EN_A4_v03.ai
IZEN_CIS2026_포스터_RU_A1_v01_작업중.psd
IZEN_회사소개영상-Full_EN_v03.prproj
IZEN_I-system_카달로그_EN_Rev03.pdf
IZEN_회사소개영상-Full_EN_Rev01.mp4
```

## 유지 규칙
- 타팀 수신 파일은 원본 파일명 유지
- 회신본만 필요 시 `_담당자v01`을 붙임
- 카메라 원본은 카메라 원본명 유지
- STEP/STL과 부품코드명은 원본 식별자 유지

## 카테고리별 예시
- 인쇄물: `브랜드_콘텐츠명_언어_규격_vNN`
- 영상 소스: `브랜드_콘텐츠명_언어_vNN`
- Google Drive 배포본: `브랜드_콘텐츠명_언어_RevNN`
- SNS: `IZEN_SNS_분류_콘텐츠_vNN`
- 렌더링 연출: `코드명_특징NN_vNN`
