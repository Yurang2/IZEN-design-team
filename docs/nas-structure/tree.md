# NAS Tree

기준: `폴더 구조` 탭의 현재 트리.

## 최상위
```text
01_PROJECT
02_ASSET
99_ARCHIVE
Google Drive
```

## 01_PROJECT
프로젝트 코드는 `IZYYNNNN_프로젝트명` 형식입니다.

```text
00_기획-문서
01_인쇄물
02_부스
03_디지털
04_영상
05_사진
06_현장수집
```

## 01_인쇄물

```text
포스터

브로슈어
카달로그
배너-현수막
certificate
```

## 04_영상
고정 폴더는 `a_자체촬영`, `b_수신`만 사용합니다.  
그 외는 영상 종류별로 생성합니다.

```text
a_자체촬영
b_수신/외주
b_수신/타팀
후기영상
티저
모션그래픽
강연소개
제품영상
회사소개영상
홍보영상
클립
```

## 05_사진
```text
a_자체촬영
b_수신/타팀
c_보정
d_공유/DAY1
d_공유/DAY2
```

## 02_ASSET
```text
01_로고
02_제품-렌더링
03_3D-소스
04_브랜드-가이드
05_폰트
06_템플릿
07_제품사진-원본
08_패키지
09_임상
연자
```

`02_제품-렌더링`은 현재 가이드의 부품유형별 구조를 유지합니다.  
`99_ARCHIVE/2024_07_이전`은 이름만 바꿔 보존하며 재분류하지 않습니다.
