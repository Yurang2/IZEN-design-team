# Folder Structure Tree

기준: `NAS 폴더 구조 가이드 > 폴더 구조` 탭의 현재 전체 트리 (`NAS + Google Drive`).

## 최상위
```text
01_PROJECT
02_ASSET
99_ARCHIVE
Google Drive
```

## 01_PROJECT
프로젝트 코드는 `IZYYNNNN_프로젝트명` 형식입니다.

```text
00_기획-문서
01_인쇄물
02_부스
03_디지털
04_영상
05_사진
06_현장수집
```

`00_기획-문서`: `기획서 / 보고서-제안서 / 품의서-지출결의서 / 미팅`  
`02_부스`: `부스디자인 / 부스그래픽`

## 04_영상
고정 폴더는 `a_자체촬영`, `b_수신`만 사용합니다.  
그 외는 영상 종류별로 생성합니다.

```text
a_자체촬영
b_수신/외주
b_수신/타팀
후기영상
티저
모션그래픽
강연소개
제품영상
회사소개영상
홍보영상
클립
```

## 05_사진

b_수신/타팀

```text

09_임상
연자
```

`01_로고`: `IZEN_CI / IAM / ZENEX / Cleanimplant / Dealer`  
`02_제품-렌더링`: `01_zenex_fixture / 02_cover_screw / 03_healing_abutment / 04_abutment / 05_zenex_kit / 06_sinus_combination_kit / 07_plazmax / 연출`  
`09_임상`: `자사-케이스 / 타사-레퍼런스`

`02_제품-렌더링`은 현재 가이드의 부품유형별 구조를 유지합니다.  
`99_ARCHIVE/2024_07_이전`은 이름만 바꿔 보존하며 재분류하지 않습니다.
